# TP Tool Suite — Landing Page

Single-page static site introducing the TP Tool Suite: five browser-based Thinking Process tools
for Theory of Constraints practitioners.

**Live:** https://tp-tools-suite.vercel.app

## What it is

The landing page explains the CRT → EC → FRT → PRT → TT workflow, links to each tool, and
hosts the four CLR facilitation prompts as downloadable files.

## Development

No build step. Open `index.html` directly in a browser, or serve locally:

```bash
npx serve .
```

## Deployment

This repo is configured for Vercel static deployment (`vercel.json`). Push to the main branch
and Vercel auto-deploys. No framework preset or build command needed — it's plain HTML/CSS.
